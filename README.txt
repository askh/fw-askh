Скрипт для настройки iptables

Скрипт содержит основные настройки для системы, которая может являться
шлюзом для доступа в Интернет из локальной сети, на которой могут быть запущены
серверы, которые могут быть доступны только из локальной сети, или и из
Интернета, в том числе серверы могут быть запущены в виртуальных машинах.

Скрипт настраивается через конфигурационный файл. Поскольку не все варианты
можно учесть, для того, чтобы избежать модификации скрипта под конкретные
нужды, в нём подключаются ряд дополнительных конфигурационных файлов, перед и
после основного функционала для цепочек INPUT, FORWARD и OUTPUT таблицы filter.
Эти конфигурационные файлы называются input_pre.cfg, input_post.cfg и аналогично
для оставшихся двух цепочек.

Для работы скрипта требуется ipset.

Распространяется под лицензией GPL 3 или (по вашему выбору) любой следующей
версии.
